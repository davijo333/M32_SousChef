# Recipes

Recipe cards and prep standards for breakfast dishes and beverages.

## Purpose

Give chefs a quick reference for how dishes are built — portions, steps, and ingredient usage that align with Kitchen Control links.

## Route

`/recipes`

## Layout (MVP scaffold)

- Grid of recipe cards per menu item
- Each card: name, short prep notes, linked ingredient summary
- Future: edit recipe, link to Kitchen Control dish editor

## Example cards

| Dish | Notes |
|------|-------|
| Bacon Egg & Cheese | Croissant + egg + cheddar + bacon |
| Iced Vanilla Latte | Espresso + milk + vanilla + ice; S/M/L scaling |
