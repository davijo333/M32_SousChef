# Promotions

Plan specials using expiring inventory and sales patterns.

## Purpose

Help owners turn waste risk into revenue — specials grounded in real stock and menu data (via chat tools and dashboard alerts).

## Route

`/promotions`

## Layout (MVP scaffold)

- List of suggested promotions (agent- or rule-generated in later stages)
- Each item: name, rationale (e.g. expiring spinach), target dishes
- Future: activate special, track uplift, push to POS (stretch)

## Example suggestions

- Spinach Omelette Weekday Special — use soon-to-expire spinach
- Iced Latte Add-on Combo — boost beverage attachment
- Breakfast Sandwich Bundle — deplete bread and bacon
